문자 포맷팅 태그 연습

b와 strong 태그

b태그는 문자를 굵고 진하게 표현합니다.

string 태그는 문자를 매우 중요한 텍스트를 표현합니다.

i(italic의 약자)와 em(emphasis의 약자) 태그

i태그는 문자에 기울어짐을 추가합니다.

em태그도 문자에 기울어짐을(강조할내용) 추가합니다.

small, mark, del 태그

small 태그는 글자를 작게작게 표현합니다.

mark 태그는 글자에 노란노란색 하이라이트를 표현합니다.

del 태그는 글자에 취소취소선을 표현합니다.
